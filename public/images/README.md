# This directory contains project images

Add your project images here. They can be:
- Hero images for projects
- Gallery images
- Thumbnail images (recommended: 800x600px or larger)

## Image Guidelines

- Format: JPG, PNG, WebP (recommended)
- Sizes: Optimize for web (compress with TinyPNG, ImageOptim, etc.)
- Naming: Use descriptive names (project-name-hero.jpg)
- Responsive: Provide multiple sizes for different screens
